# HyperSolid — 交易终端视觉方向 (Visual Direction v1)

> 方法：依据 frontend-design 技能（基于主题 → 两轮自我批判，刻意规避 AI 默认套路）。
> 概念一句话：**Electrum Terminal — 把液态的市场凝固成可掌控的固态。**

## 0. 主题锚定（Ground it）
- 主体：HyperSolid —— 移动端、非托管的 Hyperliquid 永续交易终端，带"离线也能跑策略"的 agentic 钱包。
- 受众：crypto-native 活跃交易者 + 想要自动化的高级用户；重速度、精度、信任，在手机上操作。
- 单一职责：让你（或你的 agent）快速、有把握地读懂市场并行动。
- 主体世界的素材：永续、订单簿、清算、资金费、杠杆、链上结算、签名、24/7 自主运行的 "agent"。
- 名字张力：Hyper(liquid) 液态 ↔ Solid 固态。产品把混沌、液态、不眠的市场，凝固成冷静、精密、可信的控制台；当你"冻结"（离线）时，agent 仍在安静运转。

## 1. 调色板（4–6 命名 hex + 语义色）
刻意避开"近黑 + 霓虹绿"默认。底色用**深海墨绿蓝**（呼应"流动性=深海"），品牌强调色用**银金 Electrum**（贵金属=价值/信任/固态），而**绿/红只留给涨跌语义**——品牌色与涨跌色不同色系，杜绝"品牌绿=上涨绿"的混淆，也跳出币安式霓虹绿。

- `Abyss`   #0A1217  深海墨蓝，主背景（非纯黑，有辨识度）
- `Trench`  #0F1A20  抬升表面（卡片/底部抽屉）
- `Reef`    #20303A  发丝分隔线/结构
- `Electrum`#E8C98F  银金签名色（金属渐变 → #F3E3B3）：仅用于 agent 信号 + 主 CTA + signature
- `Ice`     #EAF1F4  主文本
- `Mist`    #7E929C  次文本/标签/数据副信息
- 语义涨 `Jade` #34C98B / 语义跌 `Ember` #FF5C63 —— 仅用于价格/盈亏方向，绝不进品牌 chrome。

## 2. 字体（2+ 角色；数字是主角）
交易终端的人格在**数字**。避免 Inter 一把梭。
- 展示/标题 `Space Grotesk`（几何、有性格，克制用于标题与币种符号）
- 数据/数字 `Geist Mono`（或 JetBrains Mono）：等宽 tabular 数字，价格/订单簿/盈亏的主角；变动位用 Electrum 闪烁，小数位用 Mist 弱化。
- 正文/UI `Geist Sans`（工具角色）。
- 签名级处理：大号等宽数字、紧字距、tabular 对齐，让"数字排版"本身成为记忆点。

## 3. Signature（唯一记忆点）：Phase Pulse 相变脉冲
一条 Electrum 的"活体"指示，代表 agent 的心跳：
- agent 运行中（即便你离线）→ 顶部一道银金液态微光在缓慢"流动/呼吸"（液态）。
- agent 待命/已布防 → 收敛成一条静止的实线（固态）。
- 直接把"液态→固态"和唯一的 agentic 卖点，凝成同一个标志性元素；非装饰，承载真实状态。
- 触发自动交易/熔断时，脉冲短促亮一下并落一条审计记录。

## 5. 动效（克制、服务主题）
- Phase Pulse 环境脉冲（reduced-motion → 退化为静止实线）。
- 价格变动数字闪一下（~120ms，快而轻）。
- 下单/详情以"机械抽屉"上滑（一个被精心编排的瞬间，胜过零散特效）。

## 6. 质量底线
移动优先响应、可见键盘焦点、尊重 reduced-motion、数据对比度达 AA。

## 7. 我所冒的一个、且能自证的风险
把**品牌强调色设为银金 Electrum 而非绿色**，并把绿/红**完全让位给涨跌语义**。
- 风险：交易者对"绿色=crypto/上涨"有强关联，金色品牌可能"不够 crypto"。
- 自证：消除了几乎所有交易 App 都有的"品牌色/语义色撞色"，读数更明确；银金传达"贵金属般的价值/信任/固态"，恰好贴合 HyperSolid 把液态市场凝固成可信控制台、以及"离线 agent 守护资产"的叙事。这一步是刻意跳出币安式霓虹绿默认的关键。

## 8. 自我批判（对照 frontend-design 三大默认）
- 默认①奶油+衬线+赭石：未采用。
- 默认②近黑+霓虹绿/朱红：刻意规避——底色为深海墨蓝（非纯黑），品牌色为银金（非霓虹绿），绿/红仅作语义。
- 默认③报纸发丝栏：未采用。
- "深色交易终端"本身是功能必需（长时段、OLED、聚焦数据），在其内部以 Electrum 品牌 + Phase Pulse signature + 等宽数字人格做出差异化。
